# Bootstrap: GameDev

## How to run the desktop version

Running the source code:

```
./gradlew lwjgl3:run
```

Running the `jar` file:

```
java -jar bootstrap-{version}.jar
```

## Technologies used

- **JDK** Eclipse Temurin 11: https://adoptium.net
